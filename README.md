## About

이 저장소는 OpenLENA에서 관리하는 주요 오픈소스 소프트웨어의 보안 취약점(CVE) 정보를 담고 있습니다.
Manager 애플리케이션에서 자동으로 다운로드하여 보안 점검에 활용할 수 있습니다.
